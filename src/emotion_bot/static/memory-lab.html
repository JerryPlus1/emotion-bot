推荐测试方案

  1. 先点一次“重置用户”，确保本轮测试没有旧数据干扰。
  2. 在“写入消息”里填入你希望机器人记住的信息，比如名字、偏好、近期计划。
  3. 在“召回提问”里写一个追问，比如“你还记得我最近在准备什么吗？”。
  4. 点击“测试记忆”，观察是否命中了长期记忆或历史消息。
  5. 再点击“测试主动对话”，看主动开场是否会结合刚才输入的内容。
